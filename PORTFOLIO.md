API REST desarrollada con FastAPI para automatizar la captura, validación e integración inmediata de leads desde formularios web. El sistema sincroniza los registros en Airtable (CRM) y notifica en tiempo real en Slack mediante webhooks para agilizar el contacto del equipo de ventas. Construida siguiendo buenas prácticas de desarrollo: validación robusta de datos, manejo de errores con reintentos, suite de tests, y un modo de simulación segura (dry-run) para pruebas sin afectar servicios reales.
